PLAN B - BROKER EMAIL AUTOMATION SYSTEM
Business Proposal for Mor Logistics Manitoba Ltd
Prepared By: Ghani
Objective
Implement an automated broker outreach system that allows staff to send a simple WhatsApp
message and automatically generate, personalize, send, and track broker emails using approved
company templates.
Executive Summary
The proposed system will reduce repetitive manual work involved in broker outreach while
maintaining professional communication standards. The automation will allow Mor Logistics to
contact more brokers daily, improve response times, and maintain a centralized activity log for
management review.
How The System Works
User Input
The user sends a WhatsApp message to the automation number containing broker information and
lane details.
Example WhatsApp Message
Broker Mail: surinderpahil@amodispatch.ca
Lane: Laredo TX to Toronto ON
Date: June 3
Send mail to Surinder
Automation Workflow
WhatsApp → Claude AI → Gmail → Google Sheets → Confirmation Message
System Responsibilities
WhatsApp Business: Receives broker requests.
Claude AI: Selects and personalizes approved templates.
Gmail (sales@morlogistics.ca): Sends broker emails.
Google Sheets: Records all outreach activity.
Python Automation Engine: Connects and automates the entire process.
Software Stack
Software - Responsibility
WhatsApp Business - User input and confirmations
Claude AI - Template selection and personalization
Gmail - Email delivery
Google Sheets - Activity dashboard and reporting
Python Automation - Workflow automation and integration
Business Benefits
• Increase broker outreach volume.
• Reduce manual typing and repetitive work.
• Maintain consistent company branding.
• Track all broker communication.
• Scale sales outreach without additional staffing.
• Improve operational efficiency.
